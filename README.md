# Signatura

A modern PDF signing application built with Svelte 5, TypeScript, and Tailwind CSS. Signatura allows you to upload PDF documents, add signatures (drawn or uploaded), position them precisely on pages, and download signed documents.

## ğŸ“– How It Works

1. **Upload PDF** - Drag and drop or browse to select your PDF file
2. **Add Signature** - Draw a signature with your mouse/touch or upload an image
3. **Position** - Drag signatures to the perfect position on any page
4. **Resize** - Use the resize handle to adjust signature dimensions
5. **Download** - Get your signed PDF with all signatures embedded
